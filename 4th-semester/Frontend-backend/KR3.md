# Отчет по выполнению практических работ №13–18
**Предмет:** Фронтенд Бекенд разработка  
**Учящийся:** Веневцев Артём Андреевич

---

В данном цикле работ была проведена поэтапная трансформация стандартного веб-приложения в полнофункциональное **Progressive Web App (PWA)** с поддержкой офлайн-режима, уведомлений и real-time взаимодействия.

---

### Практическая работа №13: Офлайн-архитектура
    Реализована базовая архитектура с использованием **Service Worker**.
    Настроено кэширование статических ресурсов (HTML, CSS, JS).
    Обеспечен бесшовный переход интерфейса в офлайн-режим.

### Практическая работа №14: Манифест и визуальная идентификация

    Сконфигурирован файл `manifest.json`.
    Подготовлен набор иконок различных разрешений.
    Приложение корректно распознается браузером как устанавливаемое PWA.

### Практическая работа №15: App Shell и безопасность

     Внедрена архитектура **App Shell** (мгновенная загрузка каркаса).
     Развернут локальный **HTTPS-сервер**, необходимый для работы Service Workers и современных API.

### Практическая работа №16: Real-time и Push-уведомления

     Интегрирована библиотека **Socket.io** для двустороннего обмена данными через WebSockets.
     Реализован механизм **Push-уведомлений** с использованием протокола VAPID.

### Практическая работа №17: Расширенное взаимодействие

     Добавлен функционал планирования задач на стороне сервера.
     Реализованы кнопки действия в уведомлениях (например, функция **Snooze** — отложить).
